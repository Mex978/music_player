# Python Music Player

A CLI Music Player to save ram ;)

## Installation

1. `git clone https://github.com/Mex978/music_player.git`
2. `cd music_player`
3. `pip install pipenv`
4. `pipenv install`

## Usage

1. `pipenv run python main.py "awesome music"`
